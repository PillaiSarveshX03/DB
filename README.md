# Engineering Portfolio Website

A modern, minimal, dark-accent portfolio landing page for an engineering student. The site is intentionally static, fast, and easy to deploy on GitHub Pages, Netlify, Vercel, or any static host.

## Customize

Before deploying, update these placeholders in `index.html`:

- Your name or initials in the navigation brand.
- The hero introduction and project descriptions.
- Contact links for email, GitHub, LinkedIn, and resume.
- Project metrics and experience timeline entries.

## Run locally

Open `index.html` in a browser, or serve the folder with a static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deploy with GitHub Pages

1. Push this repository to GitHub.
2. Open the repository settings.
3. Go to **Pages**.
4. Set the source to the current branch and root folder.
5. Save, then wait for GitHub to publish the site URL.
